Feature

Feature list component with yes/no icons

Import

import { Feature } from 'mybu-expo-ui';

Description

The Feature component displays a list of features with check or cancel icons. It's perfect for showing included/excluded features in pricing plans, product comparisons, or feature lists.

Props

Prop Type Required Description
items FeatureItem[] Required Array of feature items to display
style StyleProp<ViewStyle> Optional Custom styles for each feature row
skeleton number Optional Number of skeleton rows to show

FeatureItem Structure

{
  icon?: "yes" | "no",  // Determines which icon to show
  text: string              // Feature description text
}

Usage Example

import { Feature } from 'mybu-expo-ui';

// Feature list
<Feature
  items={[
    { icon: "yes", text: "Unlimited access" },
    { icon: "yes", text: "Priority support" },
    { icon: "yes", text: "Cloud storage" },
    { icon: "no", text: "Custom branding" },
    { icon: "no", text: "API access" }
  ]}
/>

// Skeleton loading (3 rows)
<Feature
  items={[]}
  skeleton={3}
/>

Features

  • Yes Icon: Shows a green check-circle icon
  • No Icon: Shows a red cancel icon
  • Theme Colors: Uses theme.feature.yesColor and noColor
  • Skeleton Loading: Shows animated placeholders
  • Responsive: Uses responsive sizing for icons
