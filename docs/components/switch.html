Switch

Toggle switch component with label

Import

import { Switch } from 'mybu-expo-ui';

Description

The Switch component wraps React Native's Switch with additional features including labels, custom track colors, disabled states, and skeleton loading.

Props

Prop Type Required Description
value boolean Required Current switch state (on/off)
onValueChange (value: boolean) => void Required Callback when switch is toggled
trackColorFalse string Required Track color when switch is off
trackColorTrue string Required Track color when switch is on
label string Optional Label text displayed next to the switch
disabled boolean Optional Disables the switch when true
skeleton boolean Optional Shows skeleton loading state
style ViewStyle Optional Custom styles for the switch
labelStyle ViewStyle Optional Custom styles for the label

Usage Example

import React, { useState } from 'react';
import { Switch } from 'mybu-expo-ui';

const MyComponent = () => {
  const [notifications, setNotifications] = useState(true);
  const [darkMode, setDarkMode] = useState(false);

  return (
    <>
      <Switch
        label="Enable notifications"
        value={notifications}
        onValueChange={setNotifications}
        trackColorFalse="#ccc"
        trackColorTrue="#4CAF50"
      />

      <Switch
        label="Dark mode"
        value={darkMode}
        onValueChange={setDarkMode}
        trackColorFalse="#ccc"
        trackColorTrue="#2196F3"
      />
    </>
  );
};

// Skeleton loading
<Switch
  label=""
  value={false}
  onValueChange={() => {}}
  trackColorFalse="#ccc"
  trackColorTrue="#4CAF50"
  skeleton={true}
/>

Features

  • Label Support: Display descriptive text next to the switch
  • Custom Colors: Specify colors for both on and off states
  • Theme Integration: Falls back to theme colors when not specified
  • Skeleton Loading: Shimmer animation placeholder
  • Disabled State: Can disable user interaction
