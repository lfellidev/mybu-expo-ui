Container

Layout container with statusbar, header, alert, and bottom sheet support

Import

import { Container } from 'mybu-expo-ui';

Description

The Container component is a comprehensive layout wrapper that provides status bar configuration, header menu, alert system, bottom sheet modals, keyboard avoiding behavior, and optional scrolling. It's designed to be the root container for your screens.

Props

Prop Type Required Description
children React.ReactNode Required Content to render inside the container
statusbar boolean Optional Whether to show the status bar
scrollable boolean Optional Makes the content scrollable
noBackHandler boolean Optional Disables the hardware back button on Android
header HeaderProps Optional Header menu configuration
alert AlertProps Optional Alert configuration
bottomSheet BottomSheetProps Optional Bottom sheet configuration
modal React.ReactNode Optional Custom modal component
style StyleProp<ViewStyle> Optional Custom styles for the container

Usage Example

import React, { useState } from 'react';
import { Container, Text, Button } from 'mybu-expo-ui';

const MyScreen = () => {
  const [showAlert, setShowAlert] = useState(false);

  return (
    <Container
      statusbar
      scrollable
      header={{
        title: "My Screen",
        onPress: () => navigation.goBack()
      }}
      alert={{
        type: "success",
        visible: showAlert,
        text: "Operation successful!"
      }}
    >
      <Text>Your content here</Text>
      <Button
        label="Show Alert"
        onPress={() => setShowAlert(true)}
      />
    </Container>
  );
};

Features

  • StatusBar: Automatic status bar styling based on theme
  • KeyboardAvoidingView: Built-in keyboard handling for iOS
  • ScrollView: Optional scrollable content with bottom padding
  • Header Menu: Integrated header with back button
  • Alert System: Built-in alert component support
  • Bottom Sheet: Integrated bottom sheet modal
  • Back Handler: Control Android hardware back button
