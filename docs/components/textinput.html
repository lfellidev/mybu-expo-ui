TextInput

Single-line text input component

Import

import { TextInput } from 'mybu-expo-ui';

Description

The TextInput component provides a styled single-line input with label, icons, error states, password toggle, and skeleton loading. It supports various keyboard types and auto-complete options.

Props

Prop Type Required Description
value string Required Current input value
onChangeText (text: string) => void Required Callback when text changes
label string Required Label text for the input
showLabel boolean Optional Whether to display the label
placeholder string Optional Placeholder text
leftIcon React.ReactNode Optional Icon on the left side
rightIcon React.ReactNode Optional Icon on the right side
secureTextEntry boolean Optional Enables password mode
keyboardType KeyboardTypeOptions Optional Keyboard type (numeric, email, etc.)
autoCapitalize 'none' | 'sentences' | 'words' | 'characters' Optional Auto-capitalization behavior
autoComplete string Optional Auto-complete type
maxLength number Optional Maximum character limit
required boolean Optional Marks field as required
errorMessage string Optional Error message to display
disabled boolean Optional Disables the input
skeleton boolean Optional Shows skeleton loading state
onFocus () => void Optional Callback when input is focused
onBlur () => void Optional Callback when input loses focus

Usage Example

import React, { useState } from 'react';
import { TextInput, Icon } from 'mybu-expo-ui';

const MyComponent = () => {
  const [email, setEmail] = useState('');
  const [password, setPassword] = useState('');

  return (
    <>
      <TextInput
        label="Email"
        showLabel
        value={email}
        onChangeText={setEmail}
        placeholder="Enter your email"
        keyboardType="email-address"
        autoCapitalize="none"
        autoComplete="email"
        leftIcon={<Icon name="mail" />}
      />

      <TextInput
        label="Password"
        showLabel
        value={password}
        onChangeText={setPassword}
        placeholder="Enter your password"
        secureTextEntry
        autoComplete="password"
        leftIcon={<Icon name="lock" />}
      />
    </>
  );
};

// With validation
<TextInput
  label="Username"
  showLabel
  required
  value={username}
  onChangeText={setUsername}
  errorMessage={!username ? "Username is required" : undefined}
/>

// Phone number input
<TextInput
  label="Phone"
  showLabel
  value={phone}
  onChangeText={setPhone}
  keyboardType="phone-pad"
  maxLength={15}
/>

Features

  • Icon Support: Left and right icons
  • Password Mode: Secure text entry with toggle
  • Focus States: Visual feedback when focused
  • Error Display: Red border and error message
  • Keyboard Types: Numeric, email, phone, etc.
  • Skeleton Loading: Pulsing placeholder
  • Theme Integration: Theme colors and styling
