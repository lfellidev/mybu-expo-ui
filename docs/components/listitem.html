ListItem

Touchable list item component using FlatList

Import

import { ListItem } from 'mybu-expo-ui';

Description

The ListItem component renders a list of touchable items using React Native's FlatList. Each item can have a label, optional icon, and an onPress handler.

Props

Prop Type Required Description
data ListItemData[] Required Array of items to display
style StyleProp<ViewStyle> Optional Custom styles for each item container
labelStyle StyleProp<TextStyle> Optional Custom styles for the label text

ListItemData Structure

{
  label: string,           // Text to display
  onPress: () => void,     // Callback when item is pressed
  icon?: React.JSX.Element // Optional icon element
}

Usage Example

import { ListItem, Icon } from 'mybu-expo-ui';

const MyComponent = () => {
  const menuItems = [
    {
      label: "Profile",
      onPress: () => navigation.navigate('Profile'),
      icon: <Icon name="account" size={24} color="#333" />
    },
    {
      label: "Settings",
      onPress: () => navigation.navigate('Settings'),
      icon: <Icon name="cog" size={24} color="#333" />
    },
    {
      label: "Help",
      onPress: () => navigation.navigate('Help'),
      icon: <Icon name="help-circle" size={24} color="#333" />
    },
    {
      label: "Logout",
      onPress: () => handleLogout()
    }
  ];

  return (
    <ListItem data={menuItems} />
  );
};

Features

  • FlatList Based: Efficient rendering for long lists
  • Touchable Items: Each item is wrapped in TouchableOpacity
  • Icon Support: Optional icon element on the left
  • Theme Integration: Uses theme colors for background and borders
  • Unique Keys: Auto-generates unique keys using generateID
