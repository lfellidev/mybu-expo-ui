Button

Customizable button component with multiple styles

Import

import { Button } from 'mybu-expo-ui';

Description

The Button component is a highly customizable button with support for multiple types (primary, secondary, tertiary, link), loading states, countdown delays, skeleton loading, and ripple effects.

Props

Prop Type Required Description
label string Required The text to display on the button
onPress () => void Required Callback function when button is pressed
type 'primary' | 'secondary' | 'tertiary' | 'link' Optional The button style variant. Defaults to 'primary'
loading boolean Optional Shows a loading indicator when true
disabled boolean Optional Disables the button when true
delay number Optional Countdown delay in seconds before button is enabled
skeleton boolean Optional Shows a skeleton loading state
style StyleProp<ViewStyle> Optional Custom styles for the button container
labelStyle StyleProp<TextStyle> Optional Custom styles for the button label
themeStyle 'default' | 'pill' Optional Button shape style

Usage Examples

import { Button } from 'mybu-expo-ui';

// Primary button
<Button
  label="Submit"
  onPress={() => handleSubmit()}
/>

// Secondary button
<Button
  label="Cancel"
  type="secondary"
  onPress={() => handleCancel()}
/>

// Loading state
<Button
  label="Saving..."
  loading={true}
  onPress={() => {}}
/>

// With countdown delay
<Button
  label="Resend Code"
  delay={30}
  onPress={() => resendCode()}
/>

// Skeleton loading
<Button
  label=""
  skeleton={true}
  onPress={() => {}}
/>

Features

  • Ripple Effect: Primary and secondary buttons have a material-design ripple animation on press
  • Countdown Timer: Use the delay prop to show a countdown before the button becomes active
  • Loading State: Shows an ActivityIndicator when loading is true
  • Skeleton Loading: Animated skeleton placeholder for loading states
  • Pill Style: Set themeStyle="pill" for fully rounded buttons
  • Theme Integration: Automatically uses theme colors and styles
