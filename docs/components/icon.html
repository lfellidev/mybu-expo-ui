Icon

Multi-library icon component with touch support

Import

import { Icon } from 'mybu-expo-ui';

Description

The Icon component provides a unified interface for multiple icon libraries from @expo/vector-icons. It supports touchable icons and skeleton loading states.

Props

Prop Type Required Description
name string Required The name of the icon
size number Required The size of the icon in pixels
type IconType Optional Icon library to use. Defaults to MaterialCommunity
color string Optional Color of the icon
onPress () => void Optional Makes the icon touchable with this callback
disabled boolean Optional Disables touch when true
skeleton boolean Optional Shows a skeleton placeholder
style StyleProp<ViewStyle> Optional Custom styles

Supported Icon Types

  • MaterialCommunity (default)
  • Material
  • Ionicons
  • FontAwesome
  • Feather
  • AntDesign

Usage Example

import { Icon } from 'mybu-expo-ui';

// Basic icon
<Icon
  name="home"
  size={24}
  color="#333"
/>

// Touchable icon
<Icon
  name="settings"
  size={28}
  type="MaterialCommunity"
  onPress={() => openSettings()}
/>

// Different icon library
<Icon
  name="heart"
  size={24}
  type="FontAwesome"
  color="#E91E63"
/>

// Feather icons
<Icon
  name="user"
  size={24}
  type="Feather"
/>

// Skeleton loading
<Icon
  name="star"
  size={24}
  skeleton={true}
/>

Features

  • Multiple Libraries: Support for 6 icon libraries
  • Touchable: Becomes a button when onPress is provided
  • Skeleton: Circular skeleton placeholder with pulse animation
  • Disabled State: Can disable touch interactions
