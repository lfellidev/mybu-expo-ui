Alert

Display animated alert messages at the top of the screen

Import

import { Alert } from 'mybu-expo-ui';

Description

The Alert component displays animated alert messages that slide in from the top of the screen. It automatically dismisses after 5 seconds and supports three different types: success, danger, and info.

Props

Prop Type Required Description
text string Required The message text to display in the alert
visible boolean Required Controls whether the alert is visible
type 'success' | 'danger' | 'info' Optional The type of alert which determines the color scheme. Defaults to 'info'
style StyleProp<ViewStyle> Optional Custom styles to apply to the alert container

Usage Example

import React, { useState } from 'react';
import { Alert, Button } from 'mybu-expo-ui';

const MyComponent = () => {
  const [showAlert, setShowAlert] = useState(false);

  return (
    <>
      <Alert
        text="Operation completed successfully!"
        type="success"
        visible={showAlert}
      />
      <Button
        label="Show Alert"
        onPress={() => setShowAlert(true)}
      />
    </>
  );
};

Behavior

  • Slides in from top with a 500ms animation when visible becomes true
  • Automatically slides out after 5 seconds
  • Uses theme colors based on the type prop
  • Positioned absolutely at the top of the screen with z-index 999
