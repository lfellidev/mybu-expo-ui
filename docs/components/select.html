Select

Dropdown select component with bottom sheet

Import

import { Select } from 'mybu-expo-ui';

Description

The Select component provides a dropdown selection input that opens a bottom sheet with options. It supports icons, labels, custom children content, and skeleton loading states.

Props

Prop Type Required Description
icon string Required Icon name to display in the select field
label string Required Label/placeholder text for the select field
children React.JSX.Element Optional Custom content to render in the bottom sheet
onPress () => void Optional Callback when select is opened
onClose () => void Optional Callback when select is closed
showLabel boolean Optional Whether to show a label above the select
skeleton boolean Optional Shows skeleton loading state
containerStyle StyleProp<ViewStyle> Optional Custom styles for the container

Usage Example

import React, { useState } from 'react';
import { Select, ListItem } from 'mybu-expo-ui';

const MyComponent = () => {
  const [selectedCountry, setSelectedCountry] = useState('');

  const countries = [
    { label: "United States", onPress: () => setSelectedCountry('US') },
    { label: "Canada", onPress: () => setSelectedCountry('CA') },
    { label: "Mexico", onPress: () => setSelectedCountry('MX') },
    { label: "Brazil", onPress: () => setSelectedCountry('BR') }
  ];

  return (
    <Select
      icon="earth"
      label={selectedCountry || "Select a country"}
      showLabel
    >
      <ListItem data={countries} />
    </Select>
  );
};

// Skeleton loading
<Select
  icon="earth"
  label="Select"
  skeleton={true}
/>

Features

  • Bottom Sheet: Options displayed in a sliding bottom sheet
  • Icon Support: Display an icon in the select field
  • Custom Content: Render any children in the bottom sheet
  • Skeleton Loading: Animated loading placeholder
  • Theme Integration: Uses theme colors and styling
