InputSpinner

Numeric input with increase/decrease buttons

Import

import { InputSpinner } from 'mybu-expo-ui';

Description

The InputSpinner component provides a numeric input with plus and minus buttons for incrementing and decrementing values. It's commonly used for quantity selectors or numeric adjustments.

Props

Prop Type Required Description
value number Required The current numeric value
increaseOnPress () => void Required Callback when increase button is pressed
decreaseOnPress () => void Required Callback when decrease button is pressed
label string Optional Label text above the spinner
disabled boolean Optional Disables the buttons when true
skeleton boolean Optional Shows skeleton loading state
buttonStyle ViewStyle Optional Custom styles for the buttons
buttonText TextStyle Optional Custom styles for button text
valueStyle TextStyle Optional Custom styles for the value display

Usage Example

import React, { useState } from 'react';
import { InputSpinner } from 'mybu-expo-ui';

const MyComponent = () => {
  const [quantity, setQuantity] = useState(1);

  return (
    <InputSpinner
      label="Quantity"
      value={quantity}
      increaseOnPress={() => setQuantity(q => q + 1)}
      decreaseOnPress={() => setQuantity(q => Math.max(1, q - 1))}
    />
  );
};

// With min/max constraints
const ConstrainedSpinner = () => {
  const [value, setValue] = useState(5);
  const min = 0;
  const max = 10;

  return (
    <InputSpinner
      value={value}
      increaseOnPress={() => setValue(v => Math.min(max, v + 1))}
      decreaseOnPress={() => setValue(v => Math.max(min, v - 1))}
    />
  );
};

Features

  • Plus/Minus Buttons: Themed buttons with + and - symbols
  • Disabled State: Grayed out appearance when disabled
  • Skeleton Loading: Animated placeholder for loading states
  • Theme Integration: Uses theme colors and dimensions
  • Label Support: Optional label above the spinner
