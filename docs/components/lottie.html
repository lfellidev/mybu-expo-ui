Lottie

Lottie animation player with built-in animations

Import

import { Lottie } from 'mybu-expo-ui';

Description

The Lottie component plays Lottie animations using lottie-react-native. It includes a library of built-in animations and also supports custom animation files.

Props

Prop Type Required Description
width number Required Width of the animation
height number Required Height of the animation
name string Optional Name of built-in animation to play
src string | AnimationObject | { uri: string } Optional Custom animation source
loop boolean Optional Whether to loop the animation. Defaults to false
autoPlay boolean Optional Auto-play the animation. Defaults to true
skeleton boolean Optional Shows a skeleton placeholder
style StyleProp<ViewStyle> Optional Custom styles for the animation container

Usage Example

import { Lottie } from 'mybu-expo-ui';

// Built-in animation
<Lottie
  name="success"
  width={200}
  height={200}
/>

// Looping animation
<Lottie
  name="confetti"
  width={300}
  height={300}
  loop={true}
/>

// Custom animation file
<Lottie
  src={require('./my-animation.json')}
  width={200}
  height={200}
  autoPlay={true}
  loop={true}
/>

// Skeleton loading
<Lottie
  name="star"
  width={100}
  height={100}
  skeleton={true}
/>

Features

  • Built-in Library: 15 pre-loaded animations ready to use
  • Custom Animations: Load your own Lottie JSON files
  • Auto-play: Animations start automatically by default
  • Loop Control: Enable or disable animation looping
  • Skeleton Loading: Pulsing placeholder for loading states
  • Centered: Animations are centered in their container
