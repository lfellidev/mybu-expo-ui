Group

Layout wrapper component with themed padding

Import

import { Group } from 'mybu-expo-ui';

Description

The Group component is a simple layout wrapper that applies themed padding to its children. It supports different padding presets for inputs and buttons.

Props

Prop Type Required Description
children React.ReactNode Required Content to render inside the group
type 'input' | 'button' Optional Preset padding type for specific content
style StyleProp<ViewStyle> Optional Custom styles for the container

Usage Example

import { Group, TextInput, Button } from 'mybu-expo-ui';

// Default group
<Group>
  <Text>Content with default padding</Text>
</Group>

// Input group (10% top padding)
<Group type="input">
  <TextInput
    label="Email"
    value={email}
    onChangeText={setEmail}
  />
</Group>

// Button group (15% top padding)
<Group type="button">
  <Button
    label="Submit"
    onPress={handleSubmit}
  />
</Group>

Padding Presets

  • Default: Uses theme.group padding values
  • Input: 10% top padding for form inputs
  • Button: 15% top padding for action buttons
