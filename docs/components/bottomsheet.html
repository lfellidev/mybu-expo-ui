BottomSheet

Bottom sheet modal component with animation

Import

import { BottomSheet } from 'mybu-expo-ui';

Description

The BottomSheet component displays a modal that slides up from the bottom of the screen. It includes an overlay, close button, and support for primary/secondary action buttons or custom children content.

Props

Prop Type Required Description
visible boolean Required Controls whether the bottom sheet is visible
onClose () => void Required Callback function when the sheet is closed
text string Optional Text content to display in the sheet
children React.ReactNode Optional Custom content to render inside the sheet
buttons object Optional Primary and secondary button configuration

Button Object Structure

{
  primary?: {
    label?: string,
    onPress?: () => void,
    style?: StyleProp<ViewStyle>,
    labelStyle?: StyleProp<TextStyle>
  },
  secondary?: {
    label?: string,
    onPress?: () => void,
    style?: StyleProp<ViewStyle>,
    labelStyle?: StyleProp<TextStyle>
  }
}

Usage Example

import React, { useState } from 'react';
import { BottomSheet, Button } from 'mybu-expo-ui';

const MyComponent = () => {
  const [visible, setVisible] = useState(false);

  return (
    <>
      <Button
        label="Open Sheet"
        onPress={() => setVisible(true)}
      />
      <BottomSheet
        visible={visible}
        onClose={() => setVisible(false)}
        text="Are you sure you want to continue?"
        buttons={{
          primary: {
            label: "Confirm",
            onPress: () => {
              // Handle confirm
              setVisible(false);
            }
          },
          secondary: {
            label: "Cancel",
            onPress: () => setVisible(false)
          }
        }}
      />
    </>
  );
};

Features

  • Smooth slide-up animation (300ms duration)
  • Semi-transparent overlay backdrop
  • Close button in the top-right corner
  • Maximum height of 80% of the screen
  • Themed styling with customizable colors
  • Tap outside to close functionality
