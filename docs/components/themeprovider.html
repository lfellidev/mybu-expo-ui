ThemeProvider

Theme context provider for consistent styling

Import

import { ThemeProvider, useTheme } from 'mybu-expo-ui';

Description

The ThemeProvider component provides a React context for theming throughout your app. It allows you to define colors, fonts, and other styling properties that all mybu-expo-ui components will use.

Theme Props

Prop Type Required Description
children React.ReactNode Required Child components
theme ThemeProps Required Theme configuration object

ThemeProps Structure

Property Type Description
primary string Primary brand color
secondary string Secondary brand color
error string Error/danger color
text TextTheme Text styling configuration
input InputTheme Input field styling
switch SwitchTheme Switch component styling

Usage Example

import { ThemeProvider, useTheme } from 'mybu-expo-ui';

// Define your theme
const myTheme = {
  primary: '#667eea',
  secondary: '#764ba2',
  error: '#d32f2f',
  text: {
    family: 'poppins',
    weight: 400,
    color: '#333333',
  },
  input: {
    backgroundColor: '#f5f5f5',
    borderColor: '#e0e0e0',
    focusBorderColor: '#667eea',
    textColor: '#333333',
    placeholderColor: '#999999',
  },
  switch: {
    trackColorFalse: '#ccc',
    trackColorTrue: '#667eea',
  }
};

// Wrap your app
export default function App() {
  return (
    <ThemeProvider theme={myTheme}>
      <MyApp />
    </ThemeProvider>
  );
}

// Use theme in components
function MyComponent() {
  const theme = useTheme();

  return (
    <View style={{ backgroundColor: theme.primary }}>
      <Text style={{ color: theme.text.color }}>
        Themed content
      </Text>
    </View>
  );
}

useTheme Hook

Access the current theme from any component using the useTheme hook:

import { useTheme } from 'mybu-expo-ui';

function MyComponent() {
  const theme = useTheme();

  return (
    <Button
      backgroundColor={theme.primary}
      textColor={theme.text.color}
    >
      Themed Button
    </Button>
  );
}

Features

  • Centralized Styling: Define theme once, use everywhere
  • Type Safe: Full TypeScript support
  • Font Support: Configure default font family and weight
  • Input Theming: Consistent input field styling
  • Switch Theming: Custom switch colors
  • Easy Access: useTheme hook for any component
