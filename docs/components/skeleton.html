Skeleton

Loading skeleton placeholder component

Import

import { Skeleton } from 'mybu-expo-ui';

Description

The Skeleton component displays an animated placeholder while content is loading. It shows a pulsing gray box that transitions between light and dark shades.

Props

Prop Type Required Description
loading boolean Optional Whether to show the skeleton. Defaults to false
width number | string Optional Width of the skeleton. Defaults to '100%'
height number | string Optional Height of the skeleton. Defaults to 20
style any Optional Custom styles for the skeleton
children React.ReactNode Optional Content to render when not loading

Usage Example

import { Skeleton } from 'mybu-expo-ui';

// Basic skeleton
<Skeleton loading={true} width={200} height={20} />

// Full width skeleton
<Skeleton loading={true} height={40} />

// Circle skeleton
<Skeleton
  loading={true}
  width={50}
  height={50}
  style={{ borderRadius: 25 }}
/>

// With children (conditional rendering)
<Skeleton loading={isLoading} width={150} height={24}>
  <Text>Actual content</Text>
</Skeleton>

// Multiple skeletons for a list
{isLoading ? (
  <>
    <Skeleton loading width="100%" height={60} style={{ marginBottom: 10 }} />
    <Skeleton loading width="100%" height={60} style={{ marginBottom: 10 }} />
    <Skeleton loading width="100%" height={60} style={{ marginBottom: 10 }} />
  </>
) : (
  <MyList data={data} />
)}

Features

  • Pulse Animation: Smooth transition between #ddd and #f0f0f0
  • 800ms Duration: Each pulse cycle takes 800ms
  • Looping: Animation continues indefinitely while loading
  • Flexible Size: Accepts both numeric and percentage values
  • Conditional Render: Shows children when not loading
