Update

App update notification component

Import

import { Update } from 'mybu-expo-ui';
Description

The Update component checks for app updates using Expo's Updates API and displays a notification banner when updates are available. It supports both production and development builds.

Props

Prop Type Required Description
title string Optional Title text for the update banner
message string Optional Description text for the update
button string Optional Button label text
icon React.ReactNode Optional Custom icon component
backgroundColor string Optional Background color of the banner
textColor string Optional Text color
buttonColor string Optional Button background color
buttonTextColor string Optional Button text color
isDevBuild boolean Optional Indicates development build (skips update check)

Usage Example

import { Update } from 'mybu-expo-ui';

// Basic usage
<Update
  title="Update Available"
  message="A new version is ready to install"
  button="Update Now"
/>

// With custom styling
<Update
  title="New Version"
  message="Version 2.0 includes new features"
  button="Install"
  backgroundColor="#667eea"
  textColor="#ffffff"
  buttonColor="#ffffff"
  buttonTextColor="#667eea"
/>

// Development build
<Update
  title="Update"
  message="New update available"
  button="Update"
  isDevBuild={__DEV__}
/>

// With custom icon
<Update
  title="Update Ready"
  message="Tap to install the latest version"
  button="Update"
  icon={<Icon name="download" color="white" />}
/>

Features

  • Auto Detection: Checks for updates via expo-updates API
  • Banner Display: Shows notification when update available
  • One-Tap Install: Downloads and reloads the app
  • Loading State: Shows loading indicator during download
  • Dev Build Support: Can skip check in development
  • Customizable: Colors, text, and icon can be customized

How It Works

The component uses Expo's Updates API to:

  1. Check for available updates on mount
  2. Display a banner if update is available
  3. Download the update when user taps the button
  4. Reload the app to apply the update
