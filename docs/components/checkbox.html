Checkbox

Animated checkbox input component with smoke effect

Import

import { Checkbox } from 'mybu-expo-ui';

Description

The Checkbox component provides an animated checkbox with a unique smoke particle effect when toggled. It supports labels, error messages, disabled states, and skeleton loading.

Props

Prop Type Required Description
checked boolean Required The current checked state
onChange (value: boolean) => void Required Callback when the checkbox is toggled
label string Optional Label text displayed next to the checkbox
errorMessage string Optional Error message to display below the checkbox
disabled boolean Optional Disables the checkbox when true
skeleton boolean Optional Shows a skeleton loading state
style StyleProp<ViewStyle> Optional Custom styles for the container

Usage Example

import React, { useState } from 'react';
import { Checkbox } from 'mybu-expo-ui';

const MyComponent = () => {
  const [accepted, setAccepted] = useState(false);

  return (
    <Checkbox
      label="I accept the terms and conditions"
      checked={accepted}
      onChange={setAccepted}
      errorMessage={!accepted ? "You must accept the terms" : undefined}
    />
  );
};

Features

  • Smoke Effect: Unique animated smoke particles when toggled
  • Scale Animation: Smooth spring animation on check/uncheck
  • Error State: Display validation error messages
  • Skeleton Loading: Pulsing animation for loading states
  • Theme Integration: Uses theme colors for selected state
