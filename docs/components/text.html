Text

Styled text component with Google Fonts support

Import

import { Text } from 'mybu-expo-ui';

Description

The Text component provides styled text with support for multiple Google Fonts families, font weights, and skeleton loading. It automatically loads and applies the selected font family.

Props

Prop Type Required Description
children React.ReactNode Required Text content to display
family FontFamily Optional Font family to use
weight 100 | 300 | 400 | 500 | 600 | 700 Optional Font weight
skeleton boolean Optional Shows skeleton loading state
style StyleProp<TextStyle> Optional Custom styles for the text

Available Font Families

  • beVietnamPro
  • inter
  • lexend
  • nunito
  • poppins
  • quicksand
  • roboto
  • sora

Usage Example

import { Text } from 'mybu-expo-ui';

// Basic text (uses theme defaults)
<Text>Hello World</Text>

// With specific font family
<Text family="poppins">
  Poppins font text
</Text>

// With font weight
<Text family="inter" weight={700}>
  Bold Inter text
</Text>

// With custom styles
<Text
  family="roboto"
  weight={500}
  style={{
    fontSize: 24,
    color: '#667eea',
    textAlign: 'center'
  }}
>
  Styled heading
</Text>

// Skeleton loading
<Text skeleton={true}>
  Loading...
</Text>

Features

  • 8 Font Families: Pre-loaded Google Fonts
  • Weight Variants: Support for thin to bold weights
  • Auto Loading: Fonts are loaded automatically
  • Theme Integration: Uses theme text settings by default
  • Skeleton Loading: Placeholder animation while loading
