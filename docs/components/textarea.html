TextArea

Multi-line text input component

Import

import { TextArea } from 'mybu-expo-ui';

Description

The TextArea component provides a multi-line text input with label support, error messages, character limits, and skeleton loading. It's styled with theme colors and includes focus states.

Props

Prop Type Required Description
value string Required Current text value
onChangeText (text: string) => void Required Callback when text changes
label string Required Label text for the input
showLabel boolean Optional Whether to display the label
placeholder string Optional Placeholder text
height number Optional Height of the textarea. Defaults to 100
maxLength number Optional Maximum character limit
required boolean Optional Marks the field as required
errorMessage string Optional Error message to display
disabled boolean Optional Disables the input
skeleton boolean Optional Shows skeleton loading state

Usage Example

import React, { useState } from 'react';
import { TextArea } from 'mybu-expo-ui';

const MyComponent = () => {
  const [message, setMessage] = useState('');

  return (
    <TextArea
      label="Message"
      showLabel
      value={message}
      onChangeText={setMessage}
      placeholder="Enter your message..."
      height={150}
      maxLength={500}
    />
  );
};

// With validation
<TextArea
  label="Feedback"
  showLabel
  required
  value={feedback}
  onChangeText={setFeedback}
  errorMessage={!feedback ? "Feedback is required" : undefined}
/>

// Skeleton loading
<TextArea
  label="Comments"
  showLabel
  value=""
  onChangeText={() => {}}
  skeleton={true}
/>

Features

  • Multi-line: Supports multiple lines of text
  • Focus States: Visual feedback when focused
  • Error Display: Shows error border and message
  • Character Limit: Optional maxLength constraint
  • Skeleton Loading: Pulsing placeholder animation
  • Theme Integration: Uses theme colors and styling
