applicationData

Application information constants

Import

import { applicationData } from 'mybu-expo-ui';
Description

Contains information about the current application such as name, version, and build number. Data is sourced from expo-application.

Properties

Property Type Description
name string Application display name
version string Application version number
buildNumber string Build/bundle version

Usage Example

import { applicationData } from 'mybu-expo-ui';

// Display app info
<Text>{applicationData.name}</Text>
<Text>Version: {applicationData.version}</Text>
<Text>Build: {applicationData.buildNumber}</Text>

// In settings screen
<View>
  <Text>App Version</Text>
  <Text>{applicationData.version} ({applicationData.buildNumber})</Text>
</View>
