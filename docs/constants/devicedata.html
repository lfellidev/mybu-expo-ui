deviceData

Device information constants

Import

import { deviceData } from 'mybu-expo-ui';
Description

Contains device-specific information such as model, brand, OS version, and more. Data sourced from expo-device.

Properties
Property Type Description
brand string Device brand (e.g., "Apple", "Samsung")
modelName string Device model name
osName string Operating system name
osVersion string Operating system version

Usage Example

import { deviceData } from 'mybu-expo-ui';

// Display device info
<Text>Device: {deviceData.brand} {deviceData.modelName}</Text>
<Text>OS: {deviceData.osName} {deviceData.osVersion}</Text>

// Send to analytics
analytics.track('app_open', {
  device: deviceData.modelName,
  os: deviceData.osName,
  version: deviceData.osVersion
});
