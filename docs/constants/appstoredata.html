appStoreData

App store information constants

Import

import { appStoreData } from 'mybu-expo-ui';
Description

Contains information about where the app was installed from (App Store, Play Store, TestFlight, etc.).

Usage Example

import { appStoreData } from 'mybu-expo-ui';

// Check install source
if (appStoreData.store === 'testflight') {
  // Show beta features
}

// Log analytics
analytics.setUserProperty('install_source', appStoreData.store);
