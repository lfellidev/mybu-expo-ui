deviceLanguage

Device language constant

Import

import { deviceLanguage } from 'mybu-expo-ui';
Description

The device's current language setting (e.g., "en", "pt", "es"). Useful for internationalization and localization.

Type 
string

Usage Example

import { deviceLanguage } from 'mybu-expo-ui';

// Set app language based on device
const appLanguage = deviceLanguage || 'en';

// Load translations
const translations = require(`./locales/${deviceLanguage}.json`);

// Conditional content
{deviceLanguage === 'pt' ? (
  <Text>Olá!</Text>
) : (
  <Text>Hello!</Text>
)}
