Colors

Color palette constants

Import

import { Colors } from 'mybu-expo-ui';
Description

A collection of pre-defined color palettes for consistent styling across your app. Includes primary colors, neutrals, and status colors.

Usage Example

import { Colors } from 'mybu-expo-ui';

// Access color palettes
const primaryColor = Colors.primary;
const backgroundColor = Colors.background;
const textColor = Colors.text;

// Use in styles
<View style={{ backgroundColor: Colors.primary }}>
  <Text style={{ color: Colors.text }}>
    Styled text
  </Text>
</View>
