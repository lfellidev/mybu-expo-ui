constantsData

Expo constants data

Import

import { constantsData } from 'mybu-expo-ui';
Description

Provides access to Expo Constants including system fonts, device year class, manifest, and other expo-specific data.

Usage Example

import { constantsData } from 'mybu-expo-ui';

// Access expo constants
console.log(constantsData);

// Check manifest data
const manifest = constantsData.manifest;

// Get device year class
const deviceYear = constantsData.deviceYearClass;
