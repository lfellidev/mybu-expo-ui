setLowerCase

String lowercase utility function

Import

import { setLowerCase } from 'mybu-expo-ui';

Description

Converts a string to lowercase. A simple utility wrapper for string transformation.

Signature

function setLowerCase(str: string): string

Parameters

Parameter Type Description
str string The string to convert to lowercase

Returns

string - The input string converted to lowercase.

Usage Example

import { setLowerCase } from 'mybu-expo-ui';

setLowerCase('HELLO');        // "hello"
setLowerCase('Hello World');  // "hello world"
setLowerCase('MixedCase');    // "mixedcase"

// Normalize email input
const email = setLowerCase(userInput);
