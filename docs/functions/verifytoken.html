verifyToken

Token verification function

Import

import { verifyToken } from 'mybu-expo-ui';

Description

Verifies if the user's authentication token is still valid. Used for auto-login and session validation.

Signature

async function verifyToken(
  apiUrl: string,
  token: string,
  onSuccess: (data: any) => void,
  onError: (error: any) => void
): Promise<void>

Parameters

Parameter Type Description
apiUrl string The API URL
token string Auth token to verify
onSuccess (data) => void Called if token is valid
onError (error) => void Called if token is invalid

Usage Example

import { verifyToken } from 'mybu-expo-ui';

useEffect(() => {
  verifyToken(
    'https://api.example.com',
    storedToken,
    (user) => setUser(user),
    () => navigation.navigate('Login')
  );
}, []);
