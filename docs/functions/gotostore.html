goToStore

App store navigation utility

Import

import { goToStore } from 'mybu-expo-ui';
Description

Opens the appropriate app store (App Store or Play Store) for the current platform. Automatically detects iOS or Android and navigates to the corresponding store link.

Signature

function goToStore(
  iosId: string,
  androidPackage: string
): Promise<void>

Parameters

Parameter Type Description
iosId string The iOS App Store ID
androidPackage string The Android package name

Returns

Promise<void> - Resolves when the store is opened.

Usage Example

import { goToStore } from 'mybu-expo-ui';

// Rate app button
<Button 
  onPress={() => goToStore(
    '123456789',           // iOS App Store ID
    'com.example.myapp'    // Android package
  )}
>
  Rate Our App
</Button>

// In-app review prompt
const promptReview = async () => {
  await goToStore(
    'id123456789',
    'com.mycompany.app'
  );
};

Notes

  • Uses expo-linking to open external URLs
  • Automatically detects platform using Platform.OS
  • Opens the store app directly if installed
