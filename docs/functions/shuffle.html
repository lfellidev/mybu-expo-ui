shuffle

Array shuffling utility function

Import

import { shuffle } from 'mybu-expo-ui';

Description

Randomly shuffles the elements of an array using the Fisher-Yates algorithm. Returns a new array without modifying the original.

Signature

function shuffle<T>(array: T[]): T[]

Parameters

Parameter Type Description
array T[] The array to shuffle

Returns

T[] - A new array with elements in random order.

Usage Example

import { shuffle } from 'mybu-expo-ui';

const numbers = [1, 2, 3, 4, 5];
const shuffled = shuffle(numbers);
// e.g., [3, 1, 5, 2, 4]

const cards = ['A', 'K', 'Q', 'J'];
const shuffledCards = shuffle(cards);
// e.g., ['Q', 'A', 'J', 'K']

// Shuffle quiz questions
const questions = shuffle(allQuestions).slice(0, 10);

// Shuffle playlist
const randomPlaylist = shuffle(songs);
