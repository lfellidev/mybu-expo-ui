updateEmail

Email update request function

Import

import { updateEmail } from 'mybu-expo-ui';

Description

Initiates an email change request. Sends a verification code to the new email address.

Signature 
async function updateEmail(
  apiUrl: string,
  token: string,
  newEmail: string,
  onSuccess: (data: any) => void,
  onError: (error: any) => void
): Promise<void>

Usage Example

import { updateEmail } from 'mybu-expo-ui';

const handleEmailChange = async () => {
  await updateEmail(
    'https://api.example.com',
    authToken,
    newEmail,
    () => navigation.navigate('VerifyNewEmail'),
    (err) => setError(err.message)
  );
};
