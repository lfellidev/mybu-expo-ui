generateID

Random ID generator utility function

Import

import { generateID } from 'mybu-expo-ui';

Description

Generates a random alphanumeric ID string of the specified length. Useful for creating unique identifiers for list items, keys, etc.

Signature

function generateID(length: number): string

Parameters

Parameter Type Description
length number The length of the ID to generate

Returns

string - A random alphanumeric string of the specified length.

Usage Example

import { generateID } from 'mybu-expo-ui';

generateID(8);   // "aB3x9KmP"
generateID(16);  // "7Hy2bNqW4xZf0LmR"
generateID(4);   // "k3Jp"

// Use for list keys
const items = data.map(item => ({
  ...item,
  id: generateID(12)
}));
