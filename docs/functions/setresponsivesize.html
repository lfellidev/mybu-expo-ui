setResponsiveSize

Responsive size scaling utility

Import

import { setResponsiveSize } from 'mybu-expo-ui';

Description

Scales a size value based on screen dimensions to ensure consistent appearance across different device sizes. Uses a base reference of 375px width (iPhone SE/8).

Signature

function setResponsiveSize(size: number): number

Parameters

Parameter Type Description
size number The base size to scale

Returns

number - The scaled size value.

Usage Example

import { setResponsiveSize } from 'mybu-expo-ui';

// Responsive padding
const padding = setResponsiveSize(20);

// Responsive font size
const fontSize = setResponsiveSize(16);

// Use in styles
const styles = StyleSheet.create({
  container: {
    padding: setResponsiveSize(16),
    borderRadius: setResponsiveSize(8),
  },
  title: {
    fontSize: setResponsiveSize(24),
    marginBottom: setResponsiveSize(12),
  },
  icon: {
    width: setResponsiveSize(40),
    height: setResponsiveSize(40),
  }
});

Notes

  • Based on 375px reference width
  • Scales proportionally on larger/smaller screens
  • Use for consistent spacing, font sizes, and dimensions
