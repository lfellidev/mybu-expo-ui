setValue

Custom async storage setter

Import

import { setValue } from 'mybu-expo-ui';

Description

Stores a value in AsyncStorage. Automatically serializes objects to JSON.

Signature 
async function setValue(
  key: string,
  value: any
): Promise<void>

Parameters

Parameter Type Description
key string Storage key
value any Value to store

Usage Example

import { setValue } from 'mybu-expo-ui';

// Store string
await setValue('token', 'abc123');

// Store object
await setValue('user', { name: 'John', id: 1 });

// Store settings
await setValue('settings', {
  darkMode: true,
  notifications: false
});
