setPorcentageWidth

Percentage-based width calculation

Import

import { setPorcentageWidth } from 'mybu-expo-ui';

Description

Calculates a width value as a percentage of the device's screen width. Useful for responsive layouts.

Signature

function setPorcentageWidth(percentage: number): number

Parameters

Parameter Type Description
percentage number The percentage of screen width (0-100)

Returns

number - The calculated width in pixels.

Usage Example

import { setPorcentageWidth } from 'mybu-expo-ui';

// 50% of screen width
const halfWidth = setPorcentageWidth(50);

// Full screen width
const fullWidth = setPorcentageWidth(100);

// Two column layout
<View style={{ flexDirection: 'row' }}>
  <View style={{ width: setPorcentageWidth(50) }} />
  <View style={{ width: setPorcentageWidth(50) }} />
</View>

// Button with margin
<Button style={{ width: setPorcentageWidth(90) }}>
  Full Width Button
</Button>
