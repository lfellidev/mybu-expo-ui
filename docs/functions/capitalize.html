capitalize

String capitalization utility function

Import

import { capitalize } from 'mybu-expo-ui';

Description

Capitalizes the first letter of a string while keeping the rest of the string unchanged.

Signature

function capitalize(str: string): string

Parameters

Parameter Type Description
str string The string to capitalize

Returns

string - The input string with the first letter capitalized.

Usage Example

import { capitalize } from 'mybu-expo-ui';

capitalize('hello');       // "Hello"
capitalize('hello world'); // "Hello world"
capitalize('HELLO');       // "HELLO"
capitalize('h');           // "H"
capitalize('');            // ""
