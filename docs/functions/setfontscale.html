setFontScale

Font scaling utility function

Import

import { setFontScale } from 'mybu-expo-ui';

Description

Scales a font size value based on device font scale settings. Ensures text remains accessible while maintaining design proportions.

Signature

function setFontScale(size: number): number

Parameters

Parameter Type Description
size number The base font size to scale

Returns

number - The scaled font size.

Usage Example

import { setFontScale } from 'mybu-expo-ui';

// Scale font sizes
const titleSize = setFontScale(24);
const bodySize = setFontScale(16);
const smallSize = setFontScale(12);

// Use in styles
<Text style={{ fontSize: setFontScale(18) }}>
  Accessible text
</Text>
