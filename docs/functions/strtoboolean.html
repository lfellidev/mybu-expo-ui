strToBoolean

String to boolean conversion

Import

import { strToBoolean } from 'mybu-expo-ui';

Description

Converts a string value to a boolean. Useful for parsing configuration values or form inputs.

Signature

function strToBoolean(str: string): boolean

Parameters

Parameter Type Description
str string The string to convert

Returns

boolean - The boolean equivalent of the string.

Usage Example

import { strToBoolean } from 'mybu-expo-ui';

strToBoolean('true');   // true
strToBoolean('false');  // false
strToBoolean('TRUE');   // true
strToBoolean('1');      // true
strToBoolean('0');      // false
strToBoolean('yes');    // true
strToBoolean('no');     // false

// Parse config value
const isEnabled = strToBoolean(config.featureFlag);
