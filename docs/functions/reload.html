reload

App reload utility function

Import

import { reload } from 'mybu-expo-ui';

Description

Reloads the Expo app using the Updates API. This is useful for applying updates or resetting the app state.

Signature

async function reload(): Promise<void>

Returns

Promise<void> - Resolves when the reload is initiated.

Usage Example

import { reload } from 'mybu-expo-ui';

// Reload the app
await reload();

// Use with a button
<Button onPress={reload}>
  Restart App
</Button>

// Use after applying an update
const applyUpdate = async () => {
  await Updates.fetchUpdateAsync();
  await reload();
};

Notes

  • Uses expo-updates reloadAsync under the hood
  • Works in production and development builds
  • The app will fully restart, losing any unsaved state
