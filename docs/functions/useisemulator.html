useIsEmulator

Emulator detection hook

Import

import { useIsEmulator } from 'mybu-expo-ui';

Description

A React hook that detects whether the app is running on an emulator/simulator or a physical device. Uses expo-device under the hood.

Signature

function useIsEmulator(): boolean

Returns

boolean - true if running on emulator/simulator, false if on physical device.

Usage Example

import { useIsEmulator } from 'mybu-expo-ui';

const MyComponent = () => {
  const isEmulator = useIsEmulator();

  if (isEmulator) {
    return <Text>Running on Emulator</Text>;
  }

  return <Text>Running on Physical Device</Text>;
};

// Conditionally enable features
const App = () => {
  const isEmulator = useIsEmulator();

  return (
    <>
      {!isEmulator && <PushNotifications />}
      {isEmulator && <DevTools />}
    </>
  );
};

Use Cases

  • Disable push notifications on emulator
  • Enable developer tools only on emulator
  • Skip hardware-dependent features during development
  • Show debug information on emulator
